Bubble Sort Program

This page was created to show off a bit of my c++ Bubble Sort Program.
A bubble sort is a sorting algorithm that organizes a list by comparing to values that are next to each other repeatedly until it makes it to the end of the list.

The sorting function for the program
    void sortData(int counter, string name1[], string name2[])
{
	bool swap = false;
	string temp;
	for (int a = 0; a < counter; a++)
	{
		for (int b = 0; b < counter - 1; b++)
		{
			if (name1[b] > name1[b + 1])
			{
				temp = name1[b];
				name1[b] = name1[b + 1];
				name1[b + 1] = temp;
				swap = true;

				temp = name2[b];
				name2[b] = name2[b + 1];
				name2[b + 1] = temp;
				swap = true;
			}
		}
		if (swap == true)
			swap = false;
		else
			break;
	}
}

In the case of this program above, it is comparing a list of names in a different file and depending on the choice the user makes, will sort the list in alphebetical order by first name or last name.

For Example
If I have a list of Names such as the list below. 
        1. Bob Adams
        2. Luis Sanchez
        3. Zack Peter
        4. Kyle Ray
        5. Marvin Martinez
If you were to choose choice 1, it would order the list in alphebetical order by first names.
Choice 1: Ordered by First Name
        1. Bob Adams
        2. Kyle Ray
        3. Luis Sanchez
        4. Marvin Martinez
        5. Zach Peter
Or if you were to go with choice 2 it would list it in alphebetical order by last names.
Choice 2: Ordered by Last Name
        1. Adams Bob
        2. Martinez Marvin
        3. Peter Zach
        4. Ray Kyle
        5. Sanchez Luis

This is done by the variable 'name1[b]' which will hold a hame in it, then 'name1[b + 1]' will hold the name that follows right after 'name1[b]' in the list.
The program will then determine wether or not to switch the names places by comparing whether or not 'name1[b]' is greater than 'name1[b + 1]'.
If it is determined that 'name1[b]' is greater then the switch wont take place, and the counter will go up making it so that the value of
'name1[b+1]' will now be in 'name1[b]' and then the next in line of the list will take its previous place.
If 'name1[b]' is not greater than 'name1[b + 1]' then will skip trying to swap and instead just count the 'counter' up and move on to the next pair in the list.
